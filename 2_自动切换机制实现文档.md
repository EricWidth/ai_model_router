# 自动切换机制实现文档

## 一、功能概述

自动切换机制是 AI Model Router 的核心功能，负责在模型调用失败或资源耗尽时，自动切换到下一个可用模型，确保服务的连续性和可用性。

## 二、核心职责

1. 模型池管理（维护可用模型列表）
2. 健康检查（定期检测模型状态）
3. 故障检测（识别调用失败）
4. 自动切换（选择下一个模型）
5. 冷却机制（防止频繁切换）
6. 统计监控（记录切换次数和成功率）

## 三、切换逻辑

### 3.1 切换流程

```
请求到达 → 选择模型池 → 获取可用模型(按优先级) → 调用
                        ↓ 成功
                    返回结果
                        ↓ 失败/配额耗尽
              标记模型不可用 → 进入冷却期 → 切换下一个 → 重试
```

### 3.2 切换策略

#### 基于优先级的轮询（Round-Robin by Priority）

1. 按类型分组模型池（text/voice/image）
2. 按优先级排序模型（priority 值越小优先级越高）
3. 从高优先级开始尝试调用
4. 失败则切换到下一个可用的同优先级或次优先级模型

#### 健康检查策略

- **主动检查**: 定期向模型发送简单请求
- **被动检查**: 根据实际调用结果更新状态
- **指数退避**: 失败后等待时间逐渐增加

## 五、实现细节

### 5.1 模型池管理

```typescript
// src/core/model-pool.ts
export class ModelPool {
  private pools: Map<ModelType, Map<string, ModelState>>
  private switchConfig: SwitchConfig

  constructor(models: ModelConfig[], switchConfig: SwitchConfig) {
    this.pools = new Map()
    this.switchConfig = switchConfig
    this.initializePools(models)
  }

  private initializePools(models: ModelConfig[]): void {
    const typeMap = new Map<ModelType, ModelConfig[]>()

    for (const model of models) {
      const type = this.getModelType(model)
      if (!typeMap.has(type)) {
        typeMap.set(type, [])
      }
      typeMap.get(type)!.push(model)
    }

    for (const [type, typeModels] of typeMap.entries()) {
      const stateMap = new Map<string, ModelState>()
      for (const model of typeModels) {
        stateMap.set(model.name, {
          name: model.name,
          provider: model.provider,
          priority: model.priority,
          status: 'available',
          failureCount: 0,
          successCount: 0,
          totalRequests: 0
        })
      }
      this.pools.set(type, stateMap)
    }
  }

  private getModelType(model: ModelConfig): ModelType {
    if (model.name.startsWith('gpt-') || model.name.startsWith('claude-')) {
      return 'text'
    }
    if (model.name.startsWith('tts-')) {
      return 'voice'
    }
    if (model.name.startsWith('dall-') || model.name.startsWith('stable-')) {
      return 'image'
    }
    return 'text'
  }

  getAvailableModels(type: ModelType): string[] {
    const pool = this.pools.get(type)
    if (!pool) return []

    now = Date.now()

    return Array.from(pool.entries())
      .filter(([_, state]) => {
        if (state.status === 'cooling') {
          if (state.cooldownUntil && now > state.cooldownUntil) {
            state.status = 'available'
            state.cooldownUntil = undefined
            return true
          }
          return false
        }
        return state.status === 'available'
      })
      .sort((a, b) => a[1].priority - b[1].priority)
      .map(([name]) => name)
  }

  getModelState(type: ModelType, name: string): ModelState | undefined {
    const pool = this.pools.get(type)
    return pool?.get(name)
  }

  updateModelStatus(type: ModelType, name: string, status: ModelState['status']): void {
    const pool = this.pools.get(type)
    if (pool) {
      const state = pool.get(name)
      if (state) {
        state.status = status
        if (status === 'cooling') {
          state.cooldownUntil = Date.now() + this.switchConfig.cooldown
        }
      }
    }
  }

  recordSuccess(type: ModelType, name: string): void {
    const pool = this.pools.get(type)
    if (pool) {
      const state = pool.get(name)
      if (state) {
        state.successCount++
        state.totalRequests++
        state.status = 'available'
        state.cooldownUntil = undefined
      }
    }
  }

  recordFailure(type: ModelType, name: string): void {
    const pool = this.pools.get(type)
    if (pool) {
      const state = pool.get(name)
      if (state) {
        state.failureCount++
        state.totalRequests++
        state.lastFailureTime = Date.now()
        state.status = 'cooling'
        state.cooldownUntil = Date.now() + this.switchConfig.cooldown
      }
    }
  }

  getCurrentStats(): Map<ModelType, Map<string, ModelState>> {
    return this.pools
  }
}
```

### 5.2 切换策略

```typescript
// src/core/switch-strategy.ts
export class SwitchStrategy {
  constructor(
    private modelPool: ModelPool,
    private switchConfig: SwitchConfig
  ) {}

  async execute<T>(
    type: ModelType,
    executor: (modelName: string) => Promise<T>
  ): Promise<T> {
    const attemptedModels = new Set<string>()
    let lastError: Error | null = null

    for (let retry = 0; retry < this.switchConfig.maxRetries; retry++) {
      const availableModels = this.modelPool.getAvailableModels(type)
      const nextModel = availableModels.find(name => !attemptedModels.has(name))

      if (!nextModel) {
        break
      }

      attemptedModels.add(nextModel)

      try {
        this.modelPool.updateModelStatus(type, nextModel, 'available')
        const result = await executor(nextModel)
        this.modelPool.recordSuccess(type, nextModel)
        return result
      } catch (error) {
        lastError = error as Error
        console.error(`Model ${nextModel} failed:`, error)

        if (this.isRecoverableError(error)) {
          this.modelPool.recordFailure(type, nextModel)
        } else {
          throw error
        }
      }
    }

    throw new Error(
      'All models failed after retries. Last error: ' +
      lastError?.message
    )
  }

  private isRecoverableError(error: any): boolean {
    const errorMessages = [
      'quota exceeded',
      'rate limit exceeded',
      'insufficient quota',
      'service unavailable',
      'timeout',
      'connection refused'
    ]

    const errorMessage = error.message?.toLowerCase() || ''
    return errorMessages.some(msg => errorMessage.includes(msg))
  }
}
```

### 5.3 健康检查

```typescript
// src/core/health-check.ts
export class HealthCheck {
  private intervalId: NodeJS.Timeout | null = null
  private modelPool: ModelPool
  private adapters: Map<string, ModelAdapter>

  constructor(
    modelPool: ModelPool,
    adapters: Map<string, ModelAdapter>,
    config: SwitchConfig
  ) {
    this.modelPool = modelPool
    this.adapters = adapters
  }

  start(): void {
    this.intervalId = setInterval(() => {
      this.checkAllModels()
    }, this.switchConfig.healthCheckInterval)
  }

  stop(): void {
    if (this.intervalId) {
      clearInterval(this.intervalId)
      this.intervalId = null
    }
  }

  private async checkAllModels(): Promise<void> {
    const stats = this.modelPool.getCurrentStats()

    for (const [type, pool] of stats.entries()) {
      for (const [name, state] of pool.entries()) {
        if (state.status === 'unavailable' || state.status === 'cooling') {
          await this.checkModel(type, name, state)
        }
      }
    }
  }

  private async checkModel(
    type: ModelType,
    name: string,
    state: ModelState
  ): Promise<void> {
    try {
      const adapter = this.adapters.get(name)
      if (!adapter) {
        console.error(`Adapter not found for model: ${name}`)
        return
      }

      await adapter.healthCheck()

      console.log(`Health check passed for ${name}`)
      this.modelPool.recordSuccess(type, name)
    } catch (error) {
      console.error(`Health check failed for ${name}:`, error)
    }
  }
}
```

### 5.4 统计指标

```typescript
// src/core/metrics.ts
export class Metrics {
  private metrics: Map<string, ModelMetrics>
  private modelPool: ModelPool

  constructor(modelPool: ModelPool) {
    this.metrics = new Map()
    this.modelPool = modelPool
  }

  update(type: ModelType, name: string, success: boolean): void {
    const key = `${type}:${name}`
    let metric = this.metrics.get(key)

    if (!metric) {
      metric = {
        modelName: name,
        modelType: type,
        totalRequests: 0,
        successfulRequests: 0,
        failedRequests: 0,
        avgResponseTime: 0,
        lastRequestTime: Date.now(),
        uptime: 100
      }
      this.metrics.set(key, metric)
    }

    metric.totalRequests++
    metric.lastRequestTime = Date.now()

    if (success) {
      metric.successfulRequests++
    } else {
      metric.failedRequests++
    }

    metric.uptime = (metric.successfulRequests / metric.totalRequests) * 100
  }

  recordResponseTime(type: ModelType, name: string, duration: number): void {
    const key = `${type}:${name}`
    const metric = this.metrics.get(key)

    if (metric) {
      const totalResponseTime = metric.avgResponseTime * (metric.totalRequests - 1)
      metric.avgResponseTime = (totalResponseTime + duration) / metric.totalRequests
    }
  }

  getMetrics(): ModelMetrics[] {
    return Array.from(this.metrics.values())
  }

  getModelMetrics(type: ModelType, name: string): ModelMetrics | undefined {
    return this.metrics.get(`${type}:${name}`)
  }

  getOverallStats(): OverallStats {
    const allMetrics = this.getMetrics()
    const totalRequests = allMetrics.reduce((sum, m) => sum + m.totalRequests, 0)
    const totalSuccessful = allMetrics.reduce((sum, m) => sum + m.successfulRequests, 0)
    const totalFailed = allMetrics.reduce((sum, m) => sum + m.failedRequests, 0)
    const avgUptime = allMetrics.reduce((sum, m) => sum + m.uptime, 0) / allMetrics.length

    return {
      totalRequests,
      totalSuccessful,
      totalFailed,
      avgUptime,
      totalModels: allMetrics.length
    }
  }
}
```

```typescript
  it('should return available models sorted by priority', () => {
    const pool = new ModelPool([
      { name: 'model-a', provider: 'openai', priority: 2, apiKey: 'key' },
      { name: 'model-b', provider: 'openai', priority: 1, apiKey: 'key' }

    pool.recordFailure('text', 'model-a')
    const state = pool.getModelState('text', 'model-a')

    expect(state?.status).toBe('cooling')
  })
})
```

```typescript
describe('SwitchStrategy', () => {
  it('should switch to next model on failure', async () => {
    const adapter1 = { chat: async () => { throw new Error('Failed') } }
    const adapter2 = { chat: async () => ({ result: 'success' }) }

    const strategy = new SwitchStrategy(modelPool, switchConfig)

    const executor = async (name: string) => {
      const adapter = name === 'model-a' ? adapter1 : adapter2
      return await adapter.chat()
    }

    const result = await strategy.execute('text', executor)

    expect(result.result).toBe('success')
  })

  it('should throw if all models fail', async () => {
    const strategy = new SwitchStrategy(modelPool, switchConfig)

    const executor = async () => {
      throw new Error('All failed')
    }

    await expect(strategy.execute('text', executor)).rejects.toThrow('All models failed')
  })
```

## 八、性能优化

1. **模型状态缓存**: 缓存模型可用性，减少频繁检查
2. **异步健康检查**: 并行执行多个模型健康检查
3. **指数退避**: 降低健康检查频率，避免资源浪费
4. **请求批处理**: 对健康检查请求进行批量处理

## 九、扩展点

1. **自定义切换策略**: 支持负载均衡、加权随机等策略
2. **机器学习预测**: 基于历史数据预测模型可用性
3. **熔断器模式**: 快速失败，防止雪崩
4. **分布式协调**: 支持多实例之间的状态同步
5. **配置热更新**: 动态调整切换策略参数
